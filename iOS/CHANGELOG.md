# AppDimens iOS - Changelog

---

## [2.0.0] - 2025-02-01

### Added
- 13 scaling strategies (vs 2 in v1.x)
- BALANCED strategy (primary recommendation)
- LOGARITHMIC strategy (pure Weber-Fechner)
- POWER strategy (Stevens' Power Law)
- FLUID, INTERPOLATED, DIAGONAL, PERIMETER strategies
- FIT, FILL strategies (games)
- AUTOSIZE strategy (container-aware)
- Smart Inference system (18 element types)
- 5x performance improvements

### Changed
- Renamed Fixed → DEFAULT (secondary recommendation)
- Renamed Dynamic → PERCENTAGE
- Improved Metal game support

### Performance
- 5x faster cache
- Lock-free design
- Pre-calculated constants
