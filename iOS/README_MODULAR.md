# AppDimens iOS - Modular Architecture

**Version: 2.0.0**

---

## Modules

### AppDimens/Main

**Core library with 13 scaling strategies**

```ruby
pod 'AppDimens/Main', '~> 2.0.0'
```

Features:
- 13 scaling strategies
- Physical units
- Smart Inference
- Cache system

### AppDimens/UI

**UI helpers and extensions**

```ruby
pod 'AppDimens/UI', '~> 2.0.0'
```

Features:
- SwiftUI view modifiers
- UIKit extensions
- Common patterns

### AppDimens/Games

**Metal integration for game development**

```ruby
pod 'AppDimens/Games', '~> 2.0.0'
```

Features:
- Metal viewport scaling
- SIMD optimizations
- Coordinate transformations

---

## Install All

```ruby
pod 'AppDimens', '~> 2.0.0'  # Includes all modules
```
