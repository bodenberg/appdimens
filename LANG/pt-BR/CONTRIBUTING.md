## Como Contribuir

### 1. Reportar Bugs

[Abra uma issue](https://github.com/bodenberg/appdimens/issues)

### 2. Sugerir Funcionalidades

[Inicie uma discussão](https://github.com/bodenberg/appdimens/discussions)

### 3. Melhorar Documentação

- Corrigir erros de digitação
- Adicionar exemplos
- Traduzir para outros idiomas

### 4. Contribuir com Código

1. Fork o repositório
2. Crie um branch (`feature/nova-funcionalidade`)
3. Commit suas mudanças
4. Push para o branch
5. Abra um Pull Request

---

## Diretrizes

- ✅ Seguir convenções de código existentes
- ✅ Adicionar testes quando aplicável
- ✅ Atualizar documentação
- ✅ Manter compatibilidade backward
